--- layout: post title: 2022-01-20 - Legacy Set CDF files now seperate from base CDF files description: 2022-01-20 modified: 2022-01-20 category: news published: true ---

Holotable continues to support the legacy card releases. As the legacy card sets have not, and will not change, the legacy sets have been moved to seperate darkside-legacy.cdf and lightside-legacy.cdf files.

To install the legacy sets for Holotable, download the darkside-legacy.cdf and lightside-legacy.cdf files.

The darkside.cdf and lightside.cdf files will continue to have the latest release sets.
