Downloads

GTK 2

Holotableis a GTK 2 application, so you must have GTK 2 version 2.12.9, or better, installed. Windows users can download the GTK 2 installer here. (GTK 3 will not satisfy an application that requires GTK 2. Ensure that you install a version of GTK 2.)

Windows users:

New installations:

  • Download and install GTK 2
  • Download the latest Windows installer: {{ site.downloadables.windows.holotable.name }}

Upgrades:

  • Use the automatic updating capabilities of Holotable to keep up to date with program versions, CDF files, and images

Linux users:

The Linux version of holotable is out of date and not currently maintained. However, holotable can still be used on Linux thanks to Wine.

Install Wine

Install wine using apt-get or yum:

apt-get install wine

Install Holotable using Wine

wine Holotable-0.9.10-20220618-2158.exe

Run Holotable using Wine

Holotable will be installed to the virtual C: drive, which should be at ~/.wine/drive_c/Holotable. Run Holotable using the wine command:

cd ~/.wine/drive_c/Holotable wine Holotable.exe

Legacy decks

Deck Pack (2011)

kryptofis has collected some decks to give new players (particularly returning ones) something to get more in-depth with and see what is out there. These decks come from quite a few different players and have the name(s) of the author in the filename. Both Light and Dark Sides are included.
Download the deck pack (.zip file, 30KB).

Deck Pack (Virtual Block 16-era)

Psychobabble has put together a deck pack for use by players trying to get into the game. The pack contains 4 decks for each side and should provide a good selection of playable (though not necessarily dominant) decktypes as of Virtual Set 16. A readme is provided; it gives a short strategy/description for each deck.
Download the deck pack (.zip file, 107KB).
